# Master Character Prompt

## Variables
- Animal: <animal>
- Jersey Number: <number>
- Personality: <traits>
- Pose: <pose>
- Expression: <expression>

## Prompt

Create a premium AAA promotional character illustration of an anthropomorphic **<animal>** football player wearing jersey number **<number>**.

Art direction:
- Stylized semi-realistic digital painting
- Hard-edge painterly rendering
- Angular graphic shapes
- Minimal line art
- Cinematic key lighting
- Dynamic heroic pose
- High-end esports poster composition
- Premium game key art quality

Jersey:
- Modern football jersey
- Royal blue, gold, white, light blue palette
- SOCCIT centered on chest
- Original SOCCIT crest on left chest
- Matching shorts and socks
- Football boots
- Number visible

Character:
- Powerful athletic build
- Species-specific anatomy while remaining humanoid
- Expressive face
- Confident competitive attitude

Background:
- Minimal abstract geometric poster
- Large typography using player's number
- Team color accents

Negative prompt:
photorealism, messy rendering, noisy details, blurry edges, excessive accessories, watermark, logo of existing clubs, low quality.
