# SOCCIT Art Style Guide

## Overview
Create premium AAA stylized promotional character illustrations inspired by modern competitive game key art.

### Rendering
- Semi-realistic stylization
- Hard-edge painterly rendering
- Geometric shadow planes
- Minimal line art
- Bold silhouettes
- Cinematic lighting
- High readability

### Character Design
- Anthropomorphic animals
- Heroic athletic football physiques
- Dynamic poses
- Confident expressions
- Clean layered costume design

### Jersey
- Chest text: **SOCCIT**
- Left chest: original SOCCIT crest
- Use Chelsea-inspired palette only:
  - Royal Blue #001489
  - Gold #DBA111
  - White #FFFFFF
  - Light Blue #6A7AB5
  - Silver #D1D3D4
  - Small Red accents #FF0000
- Outfield players wear royal blue kit.
- Goalkeeper wears contrasting kit while keeping gold/white branding.

### Background
Minimal poster background with geometric shapes and typography.
